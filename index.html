O que é?

Lista TODO é uma lista de atividade onde se anota o que se deve fazer (to do).

Sobre

Essa é meu primeiro aplicativo para Web (Web App) e é basicamente um bloco de notas organizado em abas.
Todas as informações são mantidas no seu próprio browser, nunca saindo do seu computador. Tanto isso é verdade que tudo irá funcionar corretamente mesmo quando sem conexão com a Internet.
Funciona melhor no Firefox, com animações mais suaves e melhor suporte a teclas de atalho (claro). Também funciona no Chrome (mas claro, como ele não respeita sua privacidade ele nem vai perguntar se pode salvar coisas no seu computador). Quando ao IE, não tive tempo para esperá-lo abrir.

Atalhos de teclado

F1 exibe essa janela
Ctrl+N cria uma nova aba (não funciona no Chrome)
F2 renomea a aba ativa
Shift+Del fecha a aba ativa
Ctrl+S não faz nada (sim, tenho mania de apertar Ctrl+S a cada linha)
Ctrl+E exporta os dados
Ctrl+I permite importar dados
